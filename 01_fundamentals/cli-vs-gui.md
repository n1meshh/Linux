## What is CLI (Command Line Interface)?
CLI bhaneko **text-based interface** ho, jas ma user le commands type garcha ra system execute garcha.

Example: Terminal / Shell (bash, zsh)

### Advantages of CLI
- Lightweight, fast
- Automation-friendly (scripts)
- Full control over system
- Works over SSH (remote servers)
- Essential for DevOps

### Disadvantages of CLI
- Steep learning curve
- Not visually friendly
- Hard for beginners

---

## What is GUI (Graphical User Interface)?
GUI bhaneko **visual interface** ho, jas ma user icons, windows, buttons use garcha.

Example: Ubuntu Desktop, GNOME, KDE

### Advantages of GUI
- Easy for beginners
- Visual, user-friendly
- Drag & drop

### Disadvantages of GUI
- Heavy on resources
- Not suitable for remote servers
- Automation limited
