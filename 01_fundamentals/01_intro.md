## What is Linux?
Linux is an open-source, Unix-like operating system kernel created by Linus Torvalds.  
It is widely used in servers, cloud platforms, and DevOps environments.

Linux = Kernel  
Operating System = Kernel + Tools + Utilities

---

## Why Linux is Important for DevOps?
DevOps ko core nai Linux ho because:

- Most production servers run on Linux
- Docker and Kubernetes Linux ma based huncha
- Strong command-line support
- Easy automation using shell scripts
- Secure and stable for long-running services



## Linux vs Windows (DevOps Perspective)

| Feature       | Linux          |     Windows |
|---------------|----------------|-------------|
| Open Source   | Yes            | No          |
| Server Usage  | Very High      | Limited     |
| CLI Power     | Very Strong    | Moderate    |

## Linux Distributions
Linux different versions lai **distributions (distros)** bhanincha.

Common DevOps distros:
- Ubuntu
- CentOS / Rocky Linux
- Amazon Linux
- Debian

Each distro uses same Linux kernel but different package managers and tools.
